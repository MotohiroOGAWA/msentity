---

**msentity** is a lightweight Python toolkit for representing and manipulating  
chemical entities in mass spectrometry workflows.

It provides a unified abstraction for:

- Chemical formulas (composition, charge, exact mass)
- Molecular structures (SMILES, atom-level representation)
- Adduct ions (parsing, transformation, m/z calculation)

Designed for cheminformatics and metabolomics pipelines,  
**msentity** aims to offer a clean and consistent API for downstream analysis and modeling.
